## 🌍 DOMAIN CONNECTION

### **🔗 Connect greenafricahub.org.uk:**
1. **In Netlify Dashboard** → Site Settings → Domain Management
2. **Add custom domain**: greenafricahub.org.uk
3. **Update DNS at Namecheap**:
   ```
   Type: CNAME
   Host: @
   Value: your-site-name.netlify.app
   
   Type: CNAME
   Host: www
   Value: your-site-name.netlify.app
   ```
4. **Wait 1-24 hours** for DNS propagation
